==========

**This module is available only for iOS 7 and above.**

## COMPILING AND PACKAGING THE MODULE

Open `ios/ti.urlsession/titanium.xcconfig` and edit the Ti SDK path.

```bash
cd ios/ti.urlsession
./build.sh
```
